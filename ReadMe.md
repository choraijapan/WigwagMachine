#WigwagMachine

来自于游戏SwingCopters。

类似于flappybird，大致区别为横向改为纵向，在中间有缝隙的栅栏上增加摇摆的锤子，出边界或者碰到栅栏和锤子则游戏失败。

单击一次屏幕，控制的人物进行一次转向，每通过一个栅栏加一分。
